# About
`Ant Simulation`（アントコロニー最適化） は、スタート地点からゴールまでの最短経路を求めるアルゴリズムです。<br>
`Physarum Simulation` （粘菌ネットワーク）は、アントコロニー最適化のアルゴリズムからスタートとゴールを取り払ったアルゴリズムです。（少なくても私が実装したものはそうです）
